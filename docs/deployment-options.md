# SecureAI Gateway Deployment Options

SecureAI Gateway is designed to support enterprise AI access governance across different deployment models.

## SaaS Control Plane

The SaaS Control Plane can manage:

- Tenant configuration
- Governance policies
- Approved model routing rules
- Audit views
- Usage visibility
- Administrative workflows

## Hybrid Deployment

In a hybrid deployment, the Control Plane can manage governance configuration while the Data Plane is deployed closer to the customer environment.

This can help organizations keep sensitive-data inspection and policy execution within a controlled boundary.

## Customer-Controlled Data Plane

For higher-control environments, the Data Plane can be deployed in a customer-controlled environment.

The Data Plane can handle:

- Prompt inspection
- Sensitive-data masking
- Policy enforcement
- Model routing
- Audit event generation

This model is relevant for organizations with strict data handling, privacy or security review requirements.
