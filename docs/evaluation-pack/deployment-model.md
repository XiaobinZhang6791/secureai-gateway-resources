# SecureAI Gateway - Deployment Model

## Purpose

This document describes SecureAI Gateway deployment model options for enterprise evaluation and focused pilot planning.

## Deployment options overview

SecureAI Gateway can be discussed in SaaS Control Plane, hybrid deployment and customer-controlled Data Plane patterns.

## 1. SaaS Control Plane

The Control Plane manages policies, routing configuration, tenant metadata and audit views.

## 2. Hybrid Deployment

Hybrid deployment can combine central governance views with selected customer-controlled processing requirements.

## 3. Customer-Controlled Data Plane

The Data Plane can be deployed in a customer-controlled environment to process prompts, model outputs, sensitive-data inspection and policy enforcement.

## What the Control Plane manages

The Control Plane manages policy configuration, model routing, tenant metadata, user and department views, governance evidence and audit summaries.

## What the Data Plane processes

The Data Plane processes governed prompts, model outputs, sensitive-data inspection, masking, blocking, routing decisions and provider connectivity according to deployment scope.

## Network and integration assumptions

Network, identity provider, model provider, logging, monitoring and secret-management assumptions should be confirmed during enterprise evaluation.

## Customer and SecureAI responsibility split

SecureAI provides product guidance and evaluation support. Customers approve environment design, network access, identity integration, provider terms, operating controls and production readiness.

## Focused pilot deployment path

A focused pilot can start with sample data, a limited use case, defined policy checks and agreed evidence review criteria.

## Current limitations and non-claims

This document does not promise unimplemented integrations, regulatory approval or production certification.

## Customer responsibilities

Customers remain responsible for deployment approval, security operations, access management, data governance, monitoring, backup and legal compliance review.
